Отношение к библиотеке:
* 0 - библиотечное пространство используется как площадка для мероприятий не относящихся к основному назначению
* 3 - библиотечное пространство используется как площадка для мероприятий косвенно относящихся к основному назначению
* 4 - модернизация библиотеки
* 5 - все, что соотносится с основным назначением
